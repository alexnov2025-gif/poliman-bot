# ПОЛИМАН БОТ — Инструкция по запуску

## Локальный запуск (тест)
```
pip install python-telegram-bot==20.7
python poliman_bot.py
```

## Деплой на Railway.app (постоянная работа 24/7)

1. Зайди на https://railway.app
2. Войди через GitHub
3. New Project → Deploy from GitHub repo
4. Загрузи два файла: poliman_bot.py и requirements.txt
5. Railway автоматически установит зависимости и запустит бота

## Что умеет бот

Команды пользователя:
- /start — приветствие и меню
- 📝 Оставить заявку — запускает сбор заявки (5 шагов)
- ℹ️ О нас — информация о ПОЛИМАН
- 📞 Контакты — контакты

Шаги заявки:
1. Имя
2. Телефон
3. Размеры изделия
4. Эскиз/фото (можно пропустить)
5. Комментарий

После заполнения — заявка приходит в личку @NvrskAlexandr
с именем, телефоном, размерами, комментарием и фото эскиза (если есть).
